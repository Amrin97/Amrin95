Introduction to the Library System

This projct is supposed to be a model of a real life library management system for students. It contains data about library clients, their loans and services.


Methodology. Technologies used.

This project was made with the following technologies:

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

Work content. Project files.

This project consists of the following files:

  1. XML File
  2. XSD File
  3. XSLT file
  4. HTML file

Elements of the XML

library is the root element, it contains 5 customer records as child elements of the library.

Each record of customer has:

Account data

  • Account type
  • First name
  • Last name
  • Age
  • Library ID
  • etc.

Contact data

  • Phone
  • Email
  • Address
  • etc.

Book borrowed

  • Title
  • Author
  • ISBN
  • Year
  • Publisher
  • etc.

Detailed library account information

  • Type
  • Discount
  • Books
  • etc.

Extra library services

  • afterHoursStudyRoom
  • wifi
  • newsletter
  • sms
  • etc.

HTML table Transformation

XSLT has been used for this transformation of XML to an HTML table with customers.

Records are sorted by first name. Clients older than 18 are highlighted with green color. Those who don't have a discount are highlighted with red color.

Conclusion

This XML based project shows a possible model of a system for managing a student library. With further extra features and technologies it can be implemented to a fully working system whcih can be integrated to a real library.
